# 前言

一个前端萌新记录学习与求职的点点滴滴。同时欢迎各位前端er一起交流，我的微信：

# 主要内容

对于一名前端工程师来说，前端基础+前端框架(vue+小程序)+node+计算机网络+数据结构与算法+webpack是重中之重，也是我的核心学习路线，fighting！

# 前端基础

前端基础将包括前端三大内容：HTML(5)，CSS(3)，JavaScript(ES6)。这些基础部分的内容非常重要，毕竟如果一个前端程序员基础不牢固，那么以后在工作中肯定会收到影响。

# 前端框架

目前最主流之一也是我接触的框架便是vue了，他的数据渲染大大解放了前端er,可以更多的沉浸于业务逻辑业务，我的第一个项目便是基于vue，此外个人博客也是基于vuepress框架进行搭建。我也将继续深入vue的学习中，以及项目实战中。同时小程序开发也是必备技能。欢迎前往我的 GitHub首页 Gitee首页

# 计算机网络

计算机网络也是前端人员必备的知识点之一，特别是我这种本科为通信工程专业的学生(虽然读研转了计算机，但计网仍是不可或缺)。对于前端er，需要更加注重HTTP协议，传输层协议等计网知识学习。

# 数据结构与算法

数据结构应该算是计算机专业学生的基本功，且在求职的笔试与面试中比重越来越高。我也将重新学习，同时推荐《学习JavaScript数据结构与算法》。此外也要多在leetcode上多多刷题，毕竟code能力是每个程序员的基本功。

# node+webpack

作为前端来说，目前大前端趋势下，node与webpack也是必不可少的。

# 总结

对于现今的前端来说，技术更新很快，唯有随着前端技术的发展不断学习。博客将作为我梦想的开始，记录一枚小小前端萌新的成长之路。作为一名CS研究僧，在经历了坎坷的实习生面试后，最终来到了上海趣头条这家互联网公司，成为一名前端实习生。将通过实习日记记录三个月的熟暑期实习生活。
